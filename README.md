# Бизнес-клуб МГУ — Landing Template

## Структура проекта

- `index.html` — вся структура лендинга.
- `assets/css/styles.css` — стили, адаптивность, тёмно-синяя палитра, неоновые акценты.
- `assets/js/main.js` — интерактив:
  - мобильное меню;
  - горизонтальный скролл блока мероприятий по стрелкам;
  - всплывающая нижняя форма "Подать заявку в клуб";
  - демо-обработка форм;
  - кнопки перевода RU/EN/中文 через Google Translate.
- `assets/img/logo-placeholder.svg` — заглушка логотипа.
- `assets/img/mgu-placeholder.svg` — заглушка заставки (hero).
- `assets/img/card-placeholder.svg` — заглушка для карточек мероприятий/продуктов.

## Что заменить перед публикацией

1. Ссылки:
   - `https://t.me/your_business_club` в `index.html`.
   - все `href="#!"` на реальные страницы.
2. Изображения:
   - hero-фото МГУ: положите файл в `assets/img/mgu-hero-user.jpg` (или поменяйте путь в `index.html`);
   - логотип: SVG/PNG, формат 1:1, желательно 512x512;
   - заставка МГУ: JPG/WebP, 16:9, минимум 1920x1080;
   - карточки мероприятий: JPG/PNG, 4:3, минимум 1200x900;
   - карточки продуктов: JPG/PNG, 1:1, минимум 1000x1000.
3. Формы:
   - в `assets/js/main.js` демо-обработка без бэкенда;
   - подключите API/CRM/email-обработчик для реальной отправки.

## Примечание по автопереводу

Перевод подключён через Google Translate (`translate.google.com`).
Для работы кнопок языка на продакшене требуется доступ к внешнему интернету.
